# Анализ рынка недвижимости: Разведочный анализ данных (EDA)

## Описание проекта


Данный проект выполнен командой matplotlib в следущем составе:
- Осана https://github.com/osana
- Спиридон https://github.com/spyridonmibers
- Али https://github.com/Ali-B-Indahouse
  
Посвящен разведочному анализу данных (EDA) о рынке аренды в Москве. Цель исследования — выявить основные закономерности, тренды и аномалии в данных, а также подготовить их для дальнейшего моделирования.

**Основные задачи EDA:**
- Очистка данных от пропусков и выбросов
- Анализ распределения ключевых признаков
- Исследование взаимосвязей между характеристиками объектов
- Выявление факторов, влияющих на стоимость недвижимости
- Подготовка данных для построения предсказательных моделей

**Описание подготовки данных к анализу**
- анализ Nan значений и исключение всех столбцов где % пустых > 35%. Исключены: Название ЖК, Серия дома, Высота потолков, Мусоропровод, Площадь комнат
  при этом сохранены значения столбцов Парковка и Мусоропровод, так как анализ показывает, что значение nan тут скорее указывает на отсутствие параметра в конкретном объекте
- также исключены параметры, которые мало влияют на финальную цены или не подлежат обработке: Тип, Телефоны, Описание, Площадь комнат, м2, Окна, Санузел, Можно с детьми/животными, Дополнительно, Ссылка на объявление

## Ключевые выводы 

В ходе разведочного анализа были получены следующие результаты:

1. **Распределение цен:**  С ростом кол-ва комнат цены растут нелинейно, разброс увеличивается с увеличением кол-ва комнат. БольшАя часть предложений - 1-2 комнатные квартиры(71%)
2. **Пропуски в данных:** Обнаружены пропуски в колонках Название ЖК(75%), Серия дома(90%), Высота потолков(52%), Мусоропровод(45%), Площадь комнат(38%)
3. **Корреляции:** Сильная положительная корреляция между ценой и кол-вом комнат, ценой и площадью, площадью и кол-вом комнат и балконом и лоджией, что логично.
4. **Аномалии:** Выявлены выбросы выявлены выбросы в ценах и отрезаны по 5% 'сверху' и 'снизу' для каждой категории по кол-ву комнат
5. **Категориальные признаки:** Наиболее частые типы домов — панельный (39%) и кирпичный (22%). Большинство домов в объявлениях оборудованы лифтами 79%. Распределние по наличию парковочных мест равномерное(есть-56,6%/нет-43,4%)
